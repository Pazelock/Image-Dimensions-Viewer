# Image Dimensions Viewer
Image Dimensions Viewer Blender add-on

### About
**Image Dimensions Viewer** is a simple Blender add-on that displays image dimensions right above the image.

### Installation
1. Download a version from
   https://github.com/Pazelock/Image-Dimensions-Viewer/tree/main/Image_Dimensions_Viewer
2. Drag n' drop downloaded .zip file into Blender.
3. Enable the add-on by clicking **OK** in a pop-up.

### Usage
In **Node Editor** you will see image dimensions label above each **Image Node**. Note that label only appears above **Image Node** with selected image.
If you still don't see the labels, open the Overlays dropdown and make sure the **Image Dimensions** checkbox is set to True.
If you STILL don't see the labels try clicking the checkbox a few more times or relaunching Blender.
Follow same instructions for **Image Editor** label if you don't see it BELOW the image.

### Preferences and Personalization
To find the add-on preferences, go to:

`Edit -> Preferences -> Add-ons`

Type **"Image Dimensions Viewer"** in the search field. Click the arrow to expand the add-on information and access **Preferences**.
In the **Preferences** section, you will find several values to tweak:
  For Node Editor Overlay:
  - Font Size
  - Padding    (Backround box scale)
  - Corner Radius    (Rounded corners radius)
  - X Offset
  - Y Offset
  - Background Color
  - Text color

  For Image Editor Overlay:
  - Font Size
  - Padding
  - Corner Radius
  - Offset    (vertical only)
  - Origin    (Place the label **Below** or **Above** the image)
  - Background Color
  - Text color
